Proposta de projeto escolhida: Plataforma de música independente<br>
Breve descrição sobre seu projeto:
O Psyche é uma plataforma web voltada para artistas musicais independentes. O objetivo é oferecer um espaço para que cantores, produtores, compositores e bandas possam divulgar seu trabalho, conectar com fãs e crescer sem depender de grandes gravadoras. A home-page apresenta um cabeçalho com navegação, um banner principal, uma seção introdutória sobre a plataforma e cards com artistas em destaque.<br>


Tecnologias Utilizadas<br><br>

HTML5<br>
CSS3 (customizações visuais)<br>
Bootstrap 5.3 via CDN<br>
Bootstrap Icons 1.11 via CDN<br>
Git e GitHub<br>
